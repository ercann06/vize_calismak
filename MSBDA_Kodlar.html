Gauss Darbesi Fonksiyonu
Kaynak sinyali — geniş bantlı uyarım
Ne işe yarar?
FDTD simülasyonlarında sisteme enerji vermek için kullanılan kaynak fonksiyonudur. Matematiksel formu:
gauss = exp( −0.5 × ((t − t₀) / spread)² )
t₀ darbenin zamansal merkezi (tepe noktası), spread ise genişliğidir. Spread büyüdükçe darbe genişler → frekans bandı daralır.
ParametreTipAçıklama
tdiziZaman dizisi (örn. 1:200)
t0sayıDarbenin merkez zamanı (tepe noktası)
spreadsayıDarbenin yarı genişliği — büyük → geniş, küçük → dar
Kod 1a — Temel Gauss Darbesi
gauss_temel.m
% Temel Gauss darbesi
t = 1:1:200;
t0 = 80;         % Darbenin merkezi (80. adımda tepe)
spread = 10;     % Darbenin genişliği

A = exp(-.5*((t-t0)/spread).^2);

plot(t, A, 'b');
grid on
xlabel('Zaman', 'fontsize', 14)
ylabel('Genlik', 'fontsize', 14)
title('Gauss Darbesi', 'fontsize', 16)
set(gca, 'FontSize', 14)
Kod 1b — Farklı Spread Değerleri (Bant Genişliği Karşılaştırması)
Fiziksel yorum
spread küçük → dar darbe → geniş frekans bandı (daha fazla frekans bileşeni içerir). spread büyük → geniş darbe → dar frekans bandı.

% Farklı spread değerleri karşılaştırması
t = 1:1:200;
t0 = 80;

spread1 = 10;   % dar darbe
spread2 = 20;   % orta darbe
spread3 = 30;   % geniş darbe

A = exp(-.5*((t-t0)/spread1).^2);
B = exp(-.5*((t-t0)/spread2).^2);
C = exp(-.5*((t-t0)/spread3).^2);

plot(t,A,'b', t,B,'r', t,C,'k');
legend('spread=10', 'spread=20', 'spread=30')
grid on
set(gca,'FontSize',14)

1D FDTD — Serbest Uzay (Temel, ABC Yok)
Hard kaynak · sınırlarda tam yansıma · dalga hareketi
Güncelleme Denklemleri
E[k] = E[k] + 0.5 × ( H[k-1] − H[k] )
H[k] = H[k] + 0.5 × ( E[k] − E[k+1] )
Katsayı 0.5, Δt = Δz/(2c) seçiminden gelir (Courant sayısı = 0.5). Sınır koşulu yok → dalga duvara çarpıp geri yansır.
fdtd1d_temel.m
%--------------------------------------------------
% 1D-FDTD — Serbest Uzay — Temel (ABC Yok)
% Kaynak: Hard Gauss Darbesi
%--------------------------------------------------
clc; clear all; close all;

% --- Problem uzayı ---
ke     = 200;          % Toplam hücre sayısı
ks     = ke/2;         % Kaynak konumu (100. hücre)
nsteps = 250;          % Zaman adımı sayısı

% --- Fiziksel sabitler ---
c0 = 3e8;             % Işık hızı [m/s]
dz = 0.01;            % Hücre boyu [m]
dt = dz / (2*c0);    % Zaman adımı — Courant: dt = dz/(2c)
cc = c0*dt/dz;       % = 0.5 (normalize katsayı)

% --- Alan dizileri (başlangıçta sıfır) ---
ex = zeros(1, ke);
hy = zeros(1, ke);

% --- Gauss darbesi parametreleri ---
t0     = 20;
spread = 8;

% ========== ANA ZAMAN DÖNGÜSÜ ==========
for t = 1:nsteps

    % --- 1) Elektrik alan güncellemesi ---
    for k = 2:ke-1
        ex(k) = ex(k) + cc*(hy(k-1) - hy(k));
    end

    % --- 2) Kaynak ekleme (HARD kaynak) ---
    ex(ks) = exp(-.5*((t-t0)/spread)^2);

    % --- 3) Manyetik alan güncellemesi ---
    for k = 1:ke-1
        hy(k) = hy(k) + cc*(ex(k) - ex(k+1));
    end

    % --- Animasyon ---
    drawnow;
    plot(ex)
    xlabel('Konum', 'fontsize', 12)
    ylabel('Büyüklük', 'fontsize', 12)
    title('Ex', 'fontsize', 20)
    set(gca, 'FontSize', 20)
    axis([1 200 -2 2]); grid on
end

1D FDTD — Sol Sınır ABC
Dalga soldan geçer, sağdan yansır
ABC Prensibi
Sol sınır koşulu: E[1] = E[2] (önceki adımdan saklanır). Bu E(2) değeri iki zaman adımı boyunca saklanıp E(1)'e atanır. ex_low_m1 ve ex_low_m2 bu belleği tutar.
fdtd1d_sol_abc.m
clc; clear all; close all;

ke = 200; ks = ke/2; nsteps = 250;
c0 = 3e8; dz = 0.01; dt = dz/(2*c0); cc = c0*dt/dz;

% --- Sol sınır ABC için bellek değişkenleri ---
ex_low_m1 = 0;   % 1 adım önceki E(2)
ex_low_m2 = 0;   % 2 adım önceki E(2)

ex = zeros(1,ke); hy = zeros(1,ke);
t0 = 20; spread = 8;

for t = 1:nsteps

    for k = 2:ke-1
        ex(k) = ex(k) + cc*(hy(k-1) - hy(k));
    end

    ex(ks) = exp(-.5*((t-t0)/spread)^2);

    % ---- SOL SINIR ABC ----------------------------------------
    ex(1)     = ex_low_m2;   % E(1) ← 2 adım önceki E(2)
    ex_low_m2 = ex_low_m1;   % belleği kaydır
    ex_low_m1 = ex(2);        % mevcut E(2)'yi sakla
    % -----------------------------------------------------------

    for k = 1:ke-1
        hy(k) = hy(k) + cc*(ex(k) - ex(k+1));
    end

    drawnow; plot(ex); axis([1 200 -2 2]); grid on
end

1D FDTD — Çift Taraflı ABC (Tam Kod)
Serbest uzayda dalga sönümsüz ilerler — sınırlarda yansıma olmaz
Bu sürümde ne eklendi?
Sol sınır ABC'ye ek olarak sağ sınır ABC de eklendi: E[ke] = E[ke-1]. Dalga hem soldan hem sağdan serbest uzaya gidiyormuş gibi davranır.
fdtd1d_cift_abc.m
%--------------------------------------------------
% 1D-FDTD — Serbest Uzay — Çift Taraflı ABC
%--------------------------------------------------
clc; clear all; close all;

ke     = 200;
ks     = ke/2;
nsteps = 250;
c0     = 3e8;
dx     = 0.01;
dt     = dx / (2*c0);
cc     = c0*dt/dx;

% --- ABC bellek değişkenleri (başlangıçta sıfır) ---
ex_low_m1  = 0;  ex_low_m2  = 0;
ex_high_m1 = 0;  ex_high_m2 = 0;

ex = zeros(1, ke);
hy = zeros(1, ke);

t0     = 20;
spread = 8;

for t = 1:nsteps

    % 1) E güncellemesi
    for k = 2:ke-1
        ex(k) = ex(k) + cc*(hy(k-1) - hy(k));
    end

    % 2) Kaynak (hard)
    ex(ks) = exp(-.5*((t-t0)/spread)^2);

    % 3) SOL sınır ABC
    ex(1)      = ex_low_m2;
    ex_low_m2  = ex_low_m1;
    ex_low_m1  = ex(2);

    % 4) SAĞ sınır ABC
    ex(ke)     = ex_high_m2;
    ex_high_m2 = ex_high_m1;
    ex_high_m1 = ex(ke-1);

    % 5) H güncellemesi
    for k = 1:ke-1
        hy(k) = hy(k) + cc*(ex(k) - ex(k+1));
    end

    drawnow;
    subplot(1,2,1); plot(ex); title('Ex'); axis([1 200 -2 2]); grid on
    subplot(1,2,2); plot(hy); title('Hy'); axis([1 200 -2 2]); grid on
end

1D FDTD — Dielektrik Ortam (εᵣ ≠ 1)
cb katsayısı → 0.5/εᵣ · yansıma ve kırılma gözlenir
Temel fark
Serbest uzay: cb(k) = 0.5 · Dielektrik bölge: cb(k) = 0.5 / εᵣ
E güncellemesi: ex(k) = ex(k) + cb(k) * (hy(k-1) - hy(k))
fdtd1d_dielektrik.m
clc; clear all; close all;

ke      = 200;
ks      = 5;         % Kaynak sol tarafa yakın
nsteps  = 320;
kstart  = 100;       % Dielektrik başlangıç hücresi
kstop   = 130;       % Dielektrik bitiş hücresi
epsilon = 5;         % Bağıl dielektrik sabiti εᵣ

c0  = 3e8; dz = 0.01;
dt  = dz/(2*c0); cc = c0*dt/dz;

ex_low_m1 = 0; ex_low_m2  = 0;
ex_high_m1= 0; ex_high_m2 = 0;

ex = zeros(1,ke); hy = zeros(1,ke);
t0 = 40; spread = 12;

% --- cb katsayı dizisi ---
for k = 1:ke
    cb(k) = 0.5;             % Serbest uzay katsayısı
end
for k = kstart:kstop
    cb(k) = 0.5 / epsilon;   % Dielektrik katsayısı: 0.5/εᵣ
end

for n = 1:nsteps
    for k = 2:ke
        ex(k) = ex(k) + cb(k).*(hy(k-1) - hy(k));   % cb burada εᵣ'yi içerir
    end

    pulse1    = exp(-.5*((n-t0)/spread)^2);
    ex(ks)     = pulse1;     % Hard kaynak

    ex(1)      = ex_low_m2;  ex_low_m2  = ex_low_m1;  ex_low_m1  = ex(2);
    ex(ke)     = ex_high_m2; ex_high_m2 = ex_high_m1; ex_high_m1 = ex(ke-1);

    for k = 1:ke-1
        hy(k) = hy(k) + 0.5*(ex(k) - ex(k+1));
    end

    drawnow;
    plot(ex); xlabel('Konum'); title('Ex — Dielektrik Ortam');
    axis([1 200 -2 2]); grid on
end

1D FDTD — Sinüs Kaynak (Tek Frekans)
Kararlı hal analizi — f=700 MHz dielektrik ortam
Sinüs kaynağı formülü
pulse2 = sin(2π · f · Δt · n)
Soft kaynak olarak eklenir: ex(ks) = ex(ks) + pulse2
Dalga boyunun dielektrik içinde kısaldığını gözlemleyebilirsin.
fdtd1d_sinus_kaynak.m
clc; clear all; close all;

ke      = 200; ks = 5; nsteps = 500;
kstart  = 100; kstop = 180; epsilon = 4;

c0   = 3e8; dz = 0.01;
dt   = dz/(2*c0); cc = c0*dt/dz;
freq = 700e6;  % 700 MHz

ex_low_m1 = 0; ex_low_m2  = 0;
ex_high_m1= 0; ex_high_m2 = 0;
ex = zeros(1,ke); hy = zeros(1,ke);

for k = 1:ke;  cb(k) = 0.5; end
for k = kstart:kstop; cb(k) = 0.5/epsilon; end

for n = 1:nsteps
    for k = 2:ke
        ex(k) = ex(k) + cb(k).*(hy(k-1) - hy(k));
    end

    % SİNÜS KAYNAK — soft olarak ekleniyor
    pulse2   = sin(2*pi*freq*dt*n);
    ex(ks) = ex(ks) + pulse2;    % Soft kaynak!

    ex(1)  = ex_low_m2;  ex_low_m2  = ex_low_m1;  ex_low_m1  = ex(2);
    ex(ke)= ex_high_m2; ex_high_m2 = ex_high_m1; ex_high_m1 = ex(ke-1);

    for k = 1:ke-1
        hy(k) = hy(k) + cc*(ex(k) - ex(k+1));
    end

    drawnow; plot(ex); axis([1 200 -2 2]); grid on
    title(['Sinüs Kaynak — f=700 MHz — n=' num2str(n)])
end

1D FDTD — Modüleli Gauss Kaynak
Merkez frekans etrafında bant analizi — f_c = 5 GHz
Modüleli Gauss formülü
Modüleli Gauss = Gauss(t) × sin(2π·f_c·Δt·n)
DC bileşeni yok, enerji doğrudan f_c etrafında. pulse3 = pulse1 * pulse2
fdtd1d_moduleli_gauss.m
clc; clear all; close all;

ke      = 200; ks = 5; nsteps = 400;
kstart  = 100; kstop = 180; epsilon = 4;

c0   = 3e8; dz = 0.01;
dt   = dz/(2*c0); cc = c0*dt/dz;
freq = 5e9;       % 5 GHz merkez frekansı
t0   = 40;
spread= 12;

ex_low_m1 = 0; ex_low_m2  = 0;
ex_high_m1= 0; ex_high_m2 = 0;
ex = zeros(1,ke); hy = zeros(1,ke);

for k=1:ke;        cb(k)=0.5;           end
for k=kstart:kstop; cb(k)=0.5/epsilon; end

for n = 1:nsteps
    for k=2:ke
        ex(k)=ex(k)+cb(k).*(hy(k-1)-hy(k));
    end

    % MODÜLELI GAUSS = Gauss zarf × sinüs taşıyıcı
    pulse1 = exp(-.5*((n-t0)/spread)^2);       % Gauss zarf
    pulse2 = sin(2*pi*freq*dt*n);              % Sinüs taşıyıcı
    pulse3 = pulse1 * pulse2;                  % Modüleli Gauss
    ex(ks) = ex(ks) + pulse3;

    ex(1)  = ex_low_m2;  ex_low_m2  = ex_low_m1;  ex_low_m1  = ex(2);
    ex(ke)= ex_high_m2; ex_high_m2 = ex_high_m1; ex_high_m1 = ex(ke-1);

    for k=1:ke-1
        hy(k)=hy(k)+cc*(ex(k)-ex(k+1));
    end

    drawnow; plot(ex); axis([1 200 -2 2]); grid on
    title(['Modüleli Gauss — fc=5 GHz — n=' num2str(n)])
end

1D FDTD — Kayıplı Dielektrik Ortam (σ ≠ 0)
ca ve cb katsayıları · genlik sönümlenmesi · σ→∞ metallik duvar
Güncelleme denklemi ve katsayılar
E[k] = ca(k) · E[k] + cb(k) · (H[k-1] - H[k])
eaf = dt·σ / (2·ε₀·εᵣ)
ca(k) = (1 - eaf) / (1 + eaf)
cb(k) = 0.5 / (εᵣ · (1 + eaf))
σ=0 → ca=1, cb=0.5/εᵣ → kayıpsız dielektrik denklemine döner.
fdtd1d_kayipli.m
clc; clear all; close all;

epsz    = 8.85419e-12;  % ε₀ [F/m]
ke      = 200; ks = 5; nsteps = 450;
kstart  = 100; kstop = 200;
epsilon = 4;      % εᵣ
sigma   = 0.04;   % σ [S/m]  — σ=0 → kayıpsız, σ→1e6 → metallik

c0 = 3e8; dx = 0.01;
dt = dx/(2*c0); cc = c0*dt/dx;

ex_low_m1 =0; ex_low_m2  =0;
ex_high_m1=0; ex_high_m2 =0;
ex = zeros(1,ke); hy = zeros(1,ke);
t0 = 40; spread = 12;

% --- ca ve cb katsayıları ---
for k=1:ke
    ca(k) = 1;     % Serbest uzay: ca=1
    cb(k) = 0.5;   % Serbest uzay: cb=0.5
end

% Kayıplı dielektrik katsayıları (sigma dahil)
eaf = dt*sigma / (2*epsz*epsilon);
for k=kstart:kstop
    ca(k) = (1 - eaf) / (1 + eaf);
    cb(k) = 0.5 / (epsilon * (1 + eaf));
end

for t=1:nsteps
    for k=2:ke
        ex(k) = ca(k)*ex(k) + cb(k).*(hy(k-1)-hy(k));   % ca artık 1 değil!
    end

    pulse1   = exp(-.5*((t-t0)/spread)^2);
    ex(ks) = ex(ks) + pulse1;   % Soft Gauss

    ex(1)  = ex_low_m2;  ex_low_m2  = ex_low_m1;  ex_low_m1  = ex(2);
    ex(ke)= ex_high_m2; ex_high_m2 = ex_high_m1; ex_high_m1 = ex(ke-1);

    for k=1:ke-1
        hy(k)=hy(k)+cc*(ex(k)-ex(k+1));
    end

    drawnow; plot(ex); axis([1 200 -2 2]); grid on
    title(['Kayıplı Dielektrik (σ=0.04) — n=' num2str(t)])
end

2D FDTD — TM Modu (Ez, Hx, Hy)
60×60 ızgara · Hz=0 · merkeze Gauss darbesi · mesh + imagesc görselleştirme
TM Modu Güncelleme Denklemleri
Ez(i,j) = Ez(i,j) + 0.5·[Hy(i,j)−Hy(i−1,j)−Hx(i,j)+Hx(i,j−1)]
Hx(i,j) = Hx(i,j) − 0.5·[Ez(i,j+1) − Ez(i,j)]
Hy(i,j) = Hy(i,j) + 0.5·[Ez(i+1,j) − Ez(i,j)]
Kaynak Ez alanına ekleniyor. İç içe for döngüsü (i, j) her hücreyi günceller.
fdtd2d_TM.m
clc; clear all; close all;

IE = 60; JE = 60;    % 2D problem uzayı boyutları
is = IE/2; js = JE/2; % Kaynak merkez (30,30)
nsteps = 100;
epsz   = 8.8e-12;
c0     = 3e8; dx = 0.01;
dt     = dx/(2*c0);

% --- Alan dizilerini sıfırla (2D matrisler) ---
for j=1:JE
    for i=1:IE
        Ez(i,j)=0; Hx(i,j)=0; Hy(i,j)=0;
    end
end

to=20; w=6; T=0;

for n=1:nsteps
    T=T+1;

    % --- Ez güncellemesi ---
    for j=2:JE
        for i=2:IE
            Ez(i,j)=Ez(i,j)+0.5*[Hy(i,j)-Hy(i-1,j)-Hx(i,j)+Hx(i,j-1)];
        end
    end

    % --- Gauss darbesi — Ez alanına ekleniyor ---
    pulse = exp(-0.5*((to-T)/w)^2);
    Ez(is,js) = Ez(is,js) + pulse;   % Soft kaynak

    % --- Hx güncellemesi ---
    for j=2:JE-1
        for i=2:IE
            Hx(i,j)=Hx(i,j)+0.5*(Ez(i,j)-Ez(i,j+1));
        end
    end

    % --- Hy güncellemesi ---
    for j=2:JE
        for i=2:IE-1
            Hy(i,j)=Hy(i,j)+0.5*(Ez(i+1,j)-Ez(i,j));
        end
    end

    % --- Görselleştirme ---
    drawnow;
    [X,Y] = meshgrid(1:1:60);
    subplot(1,2,1)
    mesh(X,Y,Ez);
    xlabel('X'); ylabel('Y'); zlabel('Ez');
    set(gca,'FontSize',14)
    axis([1 60 1 60 -.25 .25]); grid on

    subplot(1,2,2)
    imagesc(Ez);       % Üstten görünüş (renk haritası)
    shading flat;
    xlabel('X'); ylabel('Y');
    set(gca,'FontSize',14)
end

2D FDTD — TE Modu (Hz, Ex, Ey)
Ez=0 · Hz alanına Gauss darbesi · ga normalizasyon matrisi
TE Modu Güncelleme Denklemleri
Ex(i,j) = Ex(i,j) + 0.5·[Hz(i,j) − Hz(i,j−1)]
Ey(i,j) = Ey(i,j) − 0.5·[Hz(i,j) − Hz(i−1,j)]
Hz(i,j) = ga(i,j)·Hz(i,j) + 0.5·[Ex(i,j+1)−Ex(i,j)−Ey(i+1,j)+Ey(i,j)]
TM'den farkı: kaynak Hz manyetik alana ekleniyor; ga matrisi normalizasyon için.
fdtd2d_TE.m
clc; clear all; close all;

IE=60; JE=60;
is=IE/2; js=JE/2;
nsteps=300;
c0=3e8; dx=0.01; dt=dx/(2*c0);

% --- Başlangıç koşulları ---
for j=1:JE
    for i=1:IE
        Hz(i,j)=0; Ex(i,j)=0; Ey(i,j)=0;
        ga(i,j)=1.0;   % Normalizasyon matrisi (serbest uzay = 1)
    end
end

to=20; w=6; T=0;

for n=1:nsteps
    T=T+1;

    % --- Ex güncellemesi ---
    for j=2:JE
        for i=2:IE
            Ex(i,j)=Ex(i,j)+0.5*(Hz(i,j)-Hz(i,j-1));
        end
    end

    % --- Ey güncellemesi ---
    for j=2:JE
        for i=2:IE
            Ey(i,j)=Ey(i,j)-0.5*(Hz(i,j)-Hz(i-1,j));
        end
    end

    % --- Hz güncellemesi ---
    for j=2:JE-1
        for i=2:IE-1
            Hz(i,j)=Hz(i,j)+0.5*[Ex(i,j+1)-Ex(i,j)-Ey(i+1,j)+Ey(i,j)];
        end
    end

    % Hz normalizasyonu (ga matrisi)
    for j=2:JE
        for i=2:IE
            Hz(i,j)=ga(i,j)*Hz(i,j);
        end
    end

    % --- Kaynak — Hz alanına ekleniyor (TM'den fark!) ---
    pulse = exp(-0.5*((to-T)/w)^2);
    Hz(is,js) = Hz(is,js) + pulse;

    drawnow;
    [X,Y] = meshgrid(1:1:60);
    subplot(1,2,1); mesh(X,Y,Hz);
    xlabel('X'); ylabel('Y'); zlabel('Hz');
    set(gca,'FontSize',14)
    axis([1 60 1 60 -.25 .25]); grid on

    subplot(1,2,2); imagesc(Hz); shading flat;
    xlabel('X'); ylabel('Y');
end

2D FDTD — Düzlem Dalga TF/SF Yöntemi
Toplam Alan / Saçılan Alan · 1D yardımcı ızgara · ia, ib, ja, jb sınırları
TF/SF prensibi
1D'de bir Gauss darbesi tanımlanır, ardından bu 1D dalga 2D uzaya enjekte edilir. Uzay iki bölgeye ayrılır: Toplam Alan (ia:ib, ja:jb içi) ve Saçılan Alan (dışı). Sınır denklemleri alanda hem çıkarma hem ekleme yapar.
fdtd2d_duzlem_dalga_TFSF.m
clc; clear all; close all;

IE=60; JE=60;
nsteps=200;
c0=3e8; dx=0.01; dt=dx/(2*c0);

% --- TF/SF sınır indeksleri ---
ia=7;  ib=IE-ia-1;   % x yönü sınırları
ja=7;  jb=JE-ja-1;   % y yönü sınırları

% --- 1D yardımcı ızgara dizileri (ABC'li) ---
ez_inc       = zeros(1,JE);
hx_inc       = zeros(1,JE);
ez_inc_low_m1=0; ez_inc_low_m2=0;
ez_inc_high_m1=0; ez_inc_high_m2=0;

% --- 2D alan dizileri ---
for j=1:JE
    for i=1:IE
        Ez(i,j)=0; Hx(i,j)=0; Hy(i,j)=0; ga(i,j)=1;
    end
end

to=20; w=6; T=0;

for n=1:nsteps
    T=T+1;

    % ===== 1D YARDIMCI IZGARA GÜNCELLEMESİ =====
    for j=2:JE
        ez_inc(j)=ez_inc(j)+0.5*(hx_inc(j-1)-hx_inc(j));
    end
    ez_inc(1)        =ez_inc_low_m2; ez_inc_low_m2 =ez_inc_low_m1; ez_inc_low_m1 =ez_inc(2);
    ez_inc(JE)       =ez_inc_high_m2; ez_inc_high_m2=ez_inc_high_m1; ez_inc_high_m1=ez_inc(JE-1);

    % ===== 2D Ez GÜNCELLEMESİ =====
    for j=2:JE
        for i=2:IE
            Ez(i,j)=Ez(i,j)+0.5*[Hy(i,j)-Hy(i-1,j)-Hx(i,j)+Hx(i,j-1)];
        end
    end

    % 1D kaynak noktası
    pulse = exp(-0.5*((to-T)/w)^2);
    ez_inc(3) = pulse;

    % TF/SF Ez modifikasyonu — alt ve üst sınırlar
    for i=ia:ib
        Ez(i,ja)=Ez(i,ja)+0.5.*hx_inc(ja-1);
        Ez(i,jb)=Ez(i,jb)-0.5.*hx_inc(jb);
    end

    % Basit sınır koşulu (köşeler ve kenarlar sıfırla)
    for j=1:JE-1
        Ez(1,j)=0; Ez(IE-1,j)=0;
    end
    for i=1:IE-1
        Ez(i,1)=0; Ez(i,JE-1)=0;
    end

    % ===== 1D Hx güncelleme =====
    for j=1:JE-1
        hx_inc(j)=hx_inc(j)+0.5.*(ez_inc(j)-ez_inc(j+1));
    end

    % ===== 2D Hx güncelleme =====
    for j=2:JE-1
        for i=2:IE-1
            Hx(i,j)=Hx(i,j)+0.5*(Ez(i,j)-Ez(i,j+1));
        end
    end
    % TF/SF Hx modifikasyonu
    for i=ia:ib
        Hx(i,ja-1)=Hx(i,ja-1)+0.5.*ez_inc(ja);
        Hx(i,jb)  =Hx(i,jb)  -0.5.*ez_inc(jb);
    end

    % ===== 2D Hy güncelleme =====
    for j=2:JE-1
        for i=2:IE-1
            Hy(i,j)=Hy(i,j)+0.5*(Ez(i+1,j)-Ez(i,j));
        end
    end
    % TF/SF Hy modifikasyonu
    for j=ja:jb
        Hy(ia-1,j)=Hy(ia-1,j)-0.5.*ez_inc(j);
        Hy(ib,j)  =Hy(ib,j)  +0.5.*ez_inc(j);
    end

    drawnow;
    [X,Y]=meshgrid(1:1:60);
    subplot(1,2,1); mesh(X,Y,Ez);
    xlabel('X'); ylabel('Y'); zlabel('Ez');
    set(gca,'FontSize',14)
    axis([1 60 1 60 -1.5 1.5]); grid on
end

Menzil çözünürlüğü
δR = c / 2B    (B: bant genişliği)
Azimut çözünürlüğü (RADAR)
δA = Rb = h·cosθ·(λ/dH)
Örn: 700km, 10 mrad → 7km çözünürlük
SAR neden üstün?
Platform hareketi boyunca çok atım → Doppler kayması → sentetik açıklık. Anten boyutu sanal olarak artırılır.
δA(SAR) = L/2    (L: fiziksel anten)
SAR azimut çözünürlüğü mesafeden BAĞIMSIZ!
SAR geometrik bozunumlar
ForeshorteningEğimli yüzey kısalmış görünür
LayoverTepe tabanından önce görünür → devrilmiş gibi
ShadowRadar ulaşamaz → karanlık bölge
LiDAR
Lazer ışığı gönder (UV/VIS/NIR/SWIR: 250nm–10μm), geri dönüşü ölç
Mesafe = c × t / 2
Yersel çözünürlük
γ ≥ 2.44λ/D  |  DI ≈ hγ
Örn: γ=1mrad, h=1km → DI≈1m
Menzil çözünürlüğü
ΔR = c·Δt / 2
Örn: 10ns darbe → min 1.5m ayrıştırma
Uyg: Ormancılık, DEM, şehir haritalama, otonom araç
SONAR
c = 1449.2 + 4.6T − 0.055T² + ... ≈ 1500 m/s
Aşağı bakan SONAR çözünürlük
δx = βH  (yatay)  |  δz = cT/2  (dikey)
Yan taramalı SONAR
δx = Rβ  |  δy = cT/(2sinθ)
Geri dönen güç ∝ 1/mesafe⁴ → yüksek güç + odaklama + CHIRP gerekir
Tarih: İlk aktif SONAR 1914 (Titanik'ten sonra)

Pankeskinleştirme
AmaçPAN (yüksek uzamsal) + MS (yüksek spektral) → her ikisi bir arada
CS (Component Substitution)
MS̃k = MS̄k + gk(P − IL)
MS uzaya dönüştür → PAN ile bileşen yer değiştir → ters dönüşüm
⚠ Spektral bozunum riski var. Global çalışır, hızlı.
CS Örnekleri
IHS, PCA, Gram-Schmidt (GS, GSA), UNB, BDSD, PRACS
IHS (Güncel adımlar)
1) MS üst-örnekle → PAN boyutuna getir
2) Bant ortalaması → I bileşeni
3) Her MS bandından I çıkar
4) Histogramı eşleştirilmiş PAN ekle
MRA (Multi-Resolution Analysis)
MS̃k = MS̄k + gk(P − PL)
(P − PL): PAN yüksek frekans bilgisi. MS'ye uzamsal detay eklenir.
✓ Daha az spektral bozunum
MRA Örnekleri
HPF, SFIM, Wavelet (WT, ATWT, AWLP), MTF-GLP, MTF-GLP-CBD
